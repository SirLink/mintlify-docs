## Configura tu entorno de desarrollo

Esta guía te ayudará a empezar con el proyecto 4Patrol. Nuestro sistema se compone de:
- Un panel administrativo basado en **Vue.js** utilizando la plantilla de producción **vue-element-admin** (con Element-UI).
- Una aplicación móvil desarrollada en **Flutter**.
- Un backend construido en **Node.js** (requiere Node.js v14 y MySQL 8).

### Clona e instala

<AccordionGroup>
  <Accordion icon="github" title="Clona el repositorio">
    Comienza clonando el repositorio del proyecto desde tu proveedor de Git:
    
    ```bash
    git clone https://github.com/tu-usuario/4patrol-template.git
    ```
    
    Una vez clonado, accede al directorio del proyecto.
  </Accordion>
  <Accordion icon="code" title="Instala las dependencias">
    Instala las dependencias necesarias para cada parte del proyecto:
    
    **Para el Panel Administrativo:**
    1. Navega al directorio `admin`:
       ```bash
       cd admin
       ```
    2. Instala las dependencias:
       ```bash
       npm install
       ```
    
    **Para la Aplicación Móvil:**
    1. Navega al directorio `flutter-app`:
       ```bash
       cd flutter-app
       ```
    2. Instala las dependencias de Flutter:
       ```bash
       flutter pub get
       ```
    
    **Para el Backend:**
    1. Navega al directorio `backend`:
       ```bash
       cd backend
       ```
    2. Instala las dependencias de Node.js:
       ```bash
       npm install
       ```
  </Accordion>
</AccordionGroup>

## Ejecuta y previsualiza tu proyecto

<AccordionGroup>
  <Accordion icon="eye" title="Inicia el Panel Administrativo">
    Ejecuta el panel administrativo basado en Vue localmente con el siguiente comando:
    
    ```bash
    npm run serve
    ```
    
    Esto iniciará un servidor de desarrollo para que puedas previsualizar la interfaz en tu navegador.
  </Accordion>
  <Accordion icon="mobile-alt" title="Ejecuta la Aplicación Móvil">
    Abre el proyecto Flutter en tu IDE preferido o ejecuta el siguiente comando:
    
    ```bash
    flutter run
    ```
    
    Este comando compilará y lanzará la aplicación móvil.
  </Accordion>
  <Accordion icon="server" title="Inicia el Backend">
    Para el backend en Node.js, asegúrate de estar utilizando Node.js v14 y ejecuta:
    
    ```bash
    node server.js
    ```
    
    Esto iniciará el servidor backend encargado del procesamiento de datos e integraciones.
  </Accordion>
</AccordionGroup>
