La aplicación móvil de 4Patrol está diseñada para ofrecer una experiencia intuitiva y eficiente en la gestión de operaciones de seguridad. A continuación, se presenta un resumen de las rutas principales de la app y sus funcionalidades:

**Rutas y Pantallas:**

- **Splash (`/`):**  
  La pantalla de bienvenida que se muestra al iniciar la aplicación. Generalmente muestra el logotipo o una animación mientras se carga la app.

- **Introducción (`/intro`):**  
  Una pantalla introductoria que explica brevemente las funcionalidades y el flujo de la aplicación.

- **Login (`/login`):**  
  Permite a los usuarios ingresar sus credenciales para acceder a la aplicación.

- **Registro (`/register`):**  
  Permite a los nuevos usuarios crear una cuenta en 4Patrol.

- **Restablecer Contraseña (`/reset_password`):**  
  Facilita la recuperación o el cambio de contraseña en caso de olvido.

- **Inicio (`/home`):**  
  La pantalla principal o dashboard donde se agrupan las funcionalidades esenciales de la app.

- **Reloj (`/clock`):**  
  Muestra el registro de tiempo, útil para el control de asistencia y seguimiento de horarios.

- **Asistente de Ronda (`/round_asistant`):**  
  Guía al usuario durante el proceso de patrullaje, asegurando el cumplimiento de los procedimientos establecidos.

- **Horarios (`/schedule`):**  
  Permite gestionar y visualizar los horarios asignados para las operaciones de seguridad.

- **Incidente (`/incident`):**  
  Facilita la notificación y registro de incidentes ocurridos durante las operaciones.

- **Reporte de Incidentes (`/incident_reports`):**  
  Muestra un listado y detalles de los incidentes reportados, permitiendo un análisis posterior.

- **Contacto (`/contact`):**  
  Facilita la comunicación con el soporte o la administración del sistema.

- **Emergencia (`/emergency`):**  
  Permite reportar situaciones de emergencia de manera rápida y efectiva.

- **Perfil (`/profile`):**  
  Permite a los usuarios ver y editar su información personal y configuraciones de cuenta.

- **Chat (`/chats`):**  
  Habilita la comunicación en tiempo real entre usuarios para mejorar la coordinación operativa.

> **Nota:** Cada ruta ha sido diseñada para cubrir necesidades específicas dentro del flujo de trabajo de 4Patrol, garantizando que los usuarios puedan gestionar de forma integral sus operaciones y comunicaciones.

Explora cada una de estas secciones para entender cómo interactuar con la aplicación móvil y aprovechar todas sus funcionalidades.
