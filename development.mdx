<Info>
  **Requisitos previos**: Asegúrate de tener instalado Node.js (v14), Flutter y MySQL 8 en tu sistema. Además, debes haber clonado los repositorios correspondientes y configurado tus archivos de entorno.
</Info>

Sigue estos pasos para configurar y ejecutar el entorno de desarrollo de 4Patrol:

## Panel Administrativo (Vue.js)

<AccordionGroup>
  <Accordion icon="github" title="Clona el repositorio del panel">
    Clona el repositorio del panel administrativo de 4Patrol:
    
    ```bash
    git clone https://github.com/tu-usuario/4patrol-admin.git
    ```
    
    Luego, accede al directorio:
    
    ```bash
    cd 4patrol-admin
    ```
  </Accordion>
  <Accordion icon="code" title="Instala las dependencias">
    Instala las dependencias necesarias ejecutando:
    
    ```bash
    npm install
    ```
  </Accordion>
  <Accordion icon="eye" title="Ejecuta el panel administrativo">
    Inicia el servidor de desarrollo con:
    
    ```bash
    npm run serve
    ```
    
    La interfaz estará disponible en `http://localhost:8080` o en el puerto configurado.
  </Accordion>
</AccordionGroup>

## Aplicación Móvil (Flutter)

<AccordionGroup>
  <Accordion icon="github" title="Clona el repositorio de la app móvil">
    Clona el repositorio de la aplicación móvil de 4Patrol:
    
    ```bash
    git clone https://github.com/tu-usuario/4patrol-mobile.git
    ```
    
    Luego, navega al directorio:
    
    ```bash
    cd 4patrol-mobile
    ```
  </Accordion>
  <Accordion icon="code" title="Instala las dependencias de Flutter">
    Instala las dependencias de Flutter con:
    
    ```bash
    flutter pub get
    ```
  </Accordion>
  <Accordion icon="mobile-alt" title="Ejecuta la aplicación móvil">
    Inicia la aplicación en un emulador o dispositivo físico:
    
    ```bash
    flutter run
    ```
  </Accordion>
</AccordionGroup>

## Backend (Node.js)

<AccordionGroup>
  <Accordion icon="github" title="Clona el repositorio del backend">
    Clona el repositorio del backend de 4Patrol:
    
    ```bash
    git clone https://github.com/tu-usuario/4patrol-backend.git
    ```
    
    Luego, accede al directorio:
    
    ```bash
    cd 4patrol-backend
    ```
  </Accordion>
  <Accordion icon="code" title="Instala las dependencias del backend">
    Instala las dependencias ejecutando:
    
    ```bash
    npm install
    ```
  </Accordion>
  <Accordion icon="server" title="Ejecuta el backend">
    Inicia el servidor backend con:
    
    ```bash
    node server.js
    ```
    
    Asegúrate de tener correctamente configurado el archivo de entorno (por ejemplo, con las credenciales de MySQL).
  </Accordion>
</AccordionGroup>
