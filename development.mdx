Sigue estos pasos para instalar y ejecutar Mintlify en tu sistema operativo:

**Paso 1**: Instala Mintlify:

<CodeGroup>

bash npm
npm i -g mintlify
bash
Copiar
yarn global add mintlify
</CodeGroup>
Paso 2: Navega al directorio de documentación (donde se encuentra el archivo docs.json) y ejecuta el siguiente comando:

bash
Copiar
mintlify dev
Una vista previa local de tu documentación estará disponible en http://localhost:3000.

Puertos Personalizados
Por defecto, Mintlify utiliza el puerto 3000. Puedes personalizar el puerto en el que se ejecuta Mintlify usando el flag --port. Por ejemplo, para ejecutar Mintlify en el puerto 3333, utiliza este comando:

bash
Copiar
mintlify dev --port 3333
Si intentas ejecutar Mintlify en un puerto que ya está en uso, usará el siguiente puerto disponible:

md
Copiar
El puerto 3000 ya está en uso. Se intentará con el 3001.
Versiones de Mintlify
Ten en cuenta que cada lanzamiento de la CLI está asociado a una versión específica de Mintlify. Si tu sitio local no coincide con la versión de producción, actualiza la CLI:

<CodeGroup>
bash
Copiar
npm i -g mintlify@latest
bash
Copiar
yarn global upgrade mintlify
</CodeGroup>
Validación de Enlaces
La CLI puede ayudarte a validar los enlaces de referencia en tu documentación. Para identificar enlaces rotos, utiliza el siguiente comando:

bash
Copiar
mintlify broken-links

Formateo de Código
Sugerimos utilizar extensiones en tu IDE para reconocer y formatear MDX. Si usas VSCode, considera la extensión MDX para VSCode para el resaltado de sintaxis, y Prettier para el formateo de código.

Resolución de Problemas
<AccordionGroup> <Accordion title='Error: No se pudo cargar el módulo "sharp" utilizando el runtime darwin-arm64'>
markdown
Copiar
Esto puede deberse a una versión obsoleta de Node. Intenta lo siguiente:
1. Elimina la versión actualmente instalada de Mintlify: `npm remove -g mintlify`
2. Actualiza a Node v19 o superior.
3. Reinstala Mintlify: `npm install -g mintlify`
</Accordion> <Accordion title="Problema: Se encuentra un error desconocido">
